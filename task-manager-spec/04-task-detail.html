Migrate auth service to JWT v2

VE-247 · Created Mar 3, 2026

Swap our current HMAC-SHA256 token signing for EdDSA signatures using the Ed25519 curve. This reduces token size by ~40% and improves verification performance.

Key changes needed:

1. Update auth-middleware.ts to use the new jose library for EdDSA verification
2. Migrate the key rotation service to generate Ed25519 key pairs
3. Add backward-compatible token validation (accept both old and new formats during migration window)
4. Update all downstream services that validate tokens directly

Migration window: 2 weeks. After that, revoke all HMAC-signed tokens.
