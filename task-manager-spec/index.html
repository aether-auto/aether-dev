Overview with stat cards, weekly activity chart, project progress bars, due-today list, and real-time activity timeline.

Stats Charts Timeline

4-column Kanban board (Backlog, Todo, In Progress, Done) with drag targets, priority tags, progress bars, and avatar stacks.

Full task view: rich description, subtask checklist with progress, comment thread, metadata sidebar, activity log, linked PRs.
